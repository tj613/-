# 卡密 PDF 阅读系统

这是一个本地可运行的网页系统，包含：

- 用户输入卡密后阅读 PDF。
- 一个卡密只绑定一个浏览器设备。
- 同一设备再次打开免输入。
- 后台生成、封禁、启用、解绑卡密并查看绑定记录。
- 阅读页禁用右键、打印、常见快捷键，并叠加动态水印。

## 启动

可以直接双击 `start-server.bat`，也可以运行：

```powershell
node server.js
```

打开：

- 用户链接：http://localhost:3000
- 后台链接：http://localhost:3000/admin

默认后台密码：

```text
admin123
```

可以用环境变量修改：

```powershell
$env:ADMIN_PASSWORD="你的新密码"; npm start
```

## 说明

网页无法 100% 禁止截屏或录屏。系统已经加入了动态水印、页面失焦遮罩、禁打印、禁右键和快捷键拦截，这些适合防普通用户传播；真正高强度防录屏需要 Windows 客户端、驱动级或 DRM 方案。

绑定设备基于浏览器本地保存的设备密钥。如果用户清空浏览器数据，会被视为新设备，需要你在后台给对应卡密解绑后重新绑定。
